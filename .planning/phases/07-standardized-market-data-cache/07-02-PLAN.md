## Phase Goal

**作为**筛选与图表消费者，**我希望**从同一 active generation 动态获得前复权数据，**从而**均线、区间高点和图表始终使用一致且可追溯的价格口径。

<objective>
实现标准化缓存读取模型，并把筛选 run 与图表绑定到精确 generation；缺失复权因子时记录跳过原因而不是降级到原始价格。

Purpose: 完成 DATA-06，并建立激活后 normalized-only 的读取边界。
Output: 动态复权 reader、筛选来源迁移、跳过记录和图表来源切换。
</objective>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| 原始行情/因子 -> 派生价格 | 缺失或异常因子可能污染筛选价格。 |
| active generation -> screening run | 筛选结果必须绑定权威数据版本。 |
| screening run -> chart | 图表必须复现同一来源，不能读取另一代缓存。 |
