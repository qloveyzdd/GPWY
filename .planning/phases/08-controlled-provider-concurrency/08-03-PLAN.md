<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Next server modules -> global runtime | 重复加载可能创建多个 scheduler。 |
| provider factory -> token-bound raw client | token 不能进入全局诊断或错误。 |
| validation -> priority metadata | 高优先级不能绕过硬并发上限。 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01 | Denial of Service | provider runtime | mitigate | globalThis 单例和跨 client 峰值集成测试。 |
| T-08-02 | Information Disclosure | runtime registry | mitigate | snapshot 不含 token、env 或 Python 路径。 |
| T-08-04 | Denial of Service | lifecycle | mitigate | close/reset 幂等并回收所有 worker。 |
| T-08-08 | Elevation of Privilege | priority | accept/mitigate | priority 仅服务端内部联合类型，validation 仍受硬上限。 |
</threat_model>
