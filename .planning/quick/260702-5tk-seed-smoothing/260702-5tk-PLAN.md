# Quick Task 260702-5tk: 对筹码模型种子做平滑扩散和重新分桶

## 目标

减少自算筹码分布中由 60 日前官方 seed 离散价位造成的孤立单柱，让 seed 进入滚动计算前先形成更连续的价格带。

## 范围

- 将筹码衰减模型版本升级到 `decay-triangle-v2`，避免复用旧模型缓存。
- 在官方 seed 完成复权调整后，对 seed 做 `0.05` 元重分桶和 `±0.20` 元三角核平滑扩散。
- 平滑后总占比仍归一为 `100%`，并继续存入 `chip_model_seed_snapshots`。
- 不修改 60 日滚动衰减公式，不修改 UI 展示逻辑。

## 验证

- 增加模型单元测试，覆盖 seed 重分桶、平滑扩散、归一化和非突变行为。
- 更新 runner 测试，确认 seed snapshot 使用平滑后的 v2 seed。
- 运行相关测试和 `npm run verify`。
