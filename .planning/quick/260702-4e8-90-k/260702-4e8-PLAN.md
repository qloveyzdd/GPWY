# Quick Task 260702-4e8: 限制筹码分布图显示到最近90日K线价格区间

## 目标

筹码分布图只使用最近最多 90 根 K 线的最高价/最低价作为可视价格范围，避免远离当前价格区间的筹码价位拉长纵轴或撑大横轴。

## 范围

- 保留官方筹码分布和计算筹码分布的原始细粒度数据。
- 仅在前端图表展示层过滤可视价位、重建 shared scale、重算可视区间内最大占比。
- 不修改筹码计算模型、缓存结构、详情 API DTO。

## 验证

- 增加组件测试覆盖 90 日价格区间外筹码不进入 y 轴、不参与 x 轴最大值和最大占比标注。
- 运行 `npm run verify`。
