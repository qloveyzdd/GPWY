# Quick Task 260702-5am: 停牌日按零换手补齐筹码计算窗口

## 目标

股票在全市场交易日停牌时，不应导致筹码计算分布显示 `missing_trade_data`。停牌日应视为无成交、无换手，筹码分布保持不变。

## 范围

- 在筹码模型专用行情读取层补齐中间停牌日。
- 对中间停牌日生成模型用虚拟 bar：价格沿用上一有效收盘价，成交量为 0，换手率为 0。
- 保持目标日缺真实 quote 时继续阻断，避免把“目标日无交易”误当成可展示的最新分布。
- 不修改 K 线图真实交易日展示，不修改缓存结构。

## 验证

- 补充 reader 单元测试覆盖中间停牌补齐和目标日停牌阻断。
- 使用 300290.SZ / 20260701 真实样本验证 on-demand 计算不再被 `20260629` 停牌日阻断。
- 运行 `npm run verify`。
