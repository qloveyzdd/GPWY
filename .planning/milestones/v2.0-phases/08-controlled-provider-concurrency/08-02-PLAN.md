<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Node -> Python stdin | 初始化消息包含 provider token。 |
| Python stdout -> Node | 不可信协议数据可能畸形、迟到或缺失 request_id。 |
| Worker lifecycle -> request Promise | error/close/timeout 可能竞态触发多次完成。 |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-02 | Information Disclosure | bridge protocol | mitigate | token 仅 stdin init；stdout/error/tests 排除 token 和路径。 |
| T-08-03 | Denial of Service | request timeout | mitigate | 超时终止进程，不保留未知状态 worker。 |
| T-08-04 | Denial of Service | worker restart | mitigate | 槽位重建预算 3、全池失效统一 reject。 |
| T-08-07 | Spoofing/Tampering | request_id | mitigate | 只接受当前槽位在途 request_id，异常协议触发 worker 重建。 |
</threat_model>
